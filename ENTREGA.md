# Entrega - Sistema de Cadastro POO II

## O que foi entregue

1. **Código-fonte completo**
   - 11 classes Java organizadas em pacotes
   - GUI com Swing (3 telas)
   - Banco de dados em memória
   - Testes JUnit

2. **Relatório Técnico** (`RELATORIO_TECNICO.md`)

## Requisitos Atendidos

✅ GUI Desktop com Swing  
✅ Back-end Java  
✅ Banco em memória  
✅ Validações: estruturas de decisão, regex, exceções  
✅ Controle de sessão e autenticação  
✅ Mensagens claras na GUI e back-end  
✅ Testes JUnit  

## Como Executar

**IntelliJ IDEA:**
1. Abra o projeto no IntelliJ
2. Clique com botão direito em `Main.java`
3. Selecione **Run 'Main.main()'**

**Maven:**
```bash
mvn clean compile
mvn exec:java
```
